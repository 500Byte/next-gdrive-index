# Product

## Register

product

## Users

Personal users managing and sharing media files (videos, images, audio) and documents. Users browse files in a clean interface to preview content before downloading. Context ranges from casual browsing on desktop to quick mobile access for media consumption.

## Product Purpose

A Google Drive directory index that simplifies file sharing by providing a clean, browsable interface for Google Drive content. Enables file preview (images, videos, audio, documents, manga) before download, with features like search, password protection, and customizable theming. The goal is to make Google Drive sharing more user-friendly and visually appealing than the default Google interface.

## Brand Personality

Clean, efficient, reliable

## Anti-references

Dark mode crypto/tech aesthetic (neon-on-black, overly technical interfaces). Avoid replicating the cluttered feel of Windows Explorer or default Google Drive. Not a generic corporate file manager.

## Design Principles

1. **Content first** - Files and media are the hero; UI should frame content, not compete with it
2. **Frictionless browsing** - Minimize clicks, provide instant previews, maintain clear navigation hierarchy
3. **Visual clarity** - Distinct file type indicators, consistent iconography, scannable file lists
4. **Performance matters** - Fast loading for media-heavy directories, efficient virtual scrolling, progressive enhancement

## Accessibility & Inclusion

WCAG 2.1 AA compliance expected. Support for:
- Reduced motion preferences (respect prefers-reduced-motion)
- Keyboard navigation for file browsing
- Screen reader friendly file lists and previews
- Color contrast ratios for text and interactive elements
